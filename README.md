# maze-solver
A simple maze solver / generator made using Python

## Controls
- `<g>`: generate a maze using Prim's Algorith
- `<c>`: clear the grid
- `<p>`: find a path either using A* or Dijkstra's
- `<s>`: set start point
- `<e>`: set end point
- `<a>`: switch to A*
- `<d>`: switch to Dijkstra's Algorithm
